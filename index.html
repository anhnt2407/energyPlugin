Introduction

This project aims to develop eclipse plugin that extends the functionality of the editor Yeti. Our goal is to make this editor enables energy aware identify instructions that can be optimized and suggesting others with lower consumption. Another feature is the ability to estimate the power consumption of the functions and application, enabling the developer to identify the instructions more energetic and have a cost estimate of the consumption of the application before being installed on the mote.
